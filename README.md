# 📅 Interactive Calendar Component (React)

A polished, interactive calendar component built using **React** and **Framer Motion**, inspired by a modern wall calendar design. This project focuses on clean UI, smooth animations, and an intuitive user experience.

---

## 🚀 Features

### 📆 Calendar Functionality

* Full **year-based navigation (2026)**
* Monthly calendar view
* **Day range selection** (start → end)
* Selection logic (auto swap if needed)

---

### 📝 Notes System

* Add notes for **selected date range**
* Notes persist using **localStorage**
* Notes are **isolated per month & date range**

---

### 🎨 UI / UX Highlights

* Wall calendar aesthetic with **dynamic hero image**
* **Month-based image switching**
* Clean and minimal layout inspired by real calendars
* Fully responsive design (desktop + mobile)

---

### ✨ Animations (Framer Motion)

* Smooth **month transition (slide animation)**
* Image **fade + zoom effect**
* Interactive **date hover & click animations**

---

### 🌟 Extra Enhancements

* ✅ Weekend highlighting (Saturday & Sunday)
* ✅ Today’s date highlight
* ✅ Smooth hover interactions
* ✅ Consistent grid layout (fixed height)

---

## 🛠 Tech Stack

* **React (Functional Components + Hooks)**
* **Framer Motion** (for animations)
* **CSS** (custom styling, responsive design)
* **LocalStorage** (data persistence)
